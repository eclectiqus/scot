Scot::Model::Entity

Description

Entitytype tracks the user defined entity types in the future we may store data about entity types in here too

Extends

Scot::Model

Consumed Roles

    Meerkat::Role::Document
    Scot::Role::Entriable
    Scot::Role::Hashable
    Scot::Role::Times

Attributes

value

The name of the EntityType, e.g. "ipaddr", "threat_group", etc.

status

valid statuses = active | disabled

order

help us sort the order to look for entity types

match

The regular expression text to match an entity

options

options to pass to SCOT's entity extractor typically { multiword => yes|no }

data

hold data in a key value store about the entity type. ideas for this include cached counts
