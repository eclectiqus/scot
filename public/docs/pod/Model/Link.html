Name

Scot::Model::Link

Description

The model of a Link Record

SCOT will allow you to associate just about anything with anything else. A link is an "edge" in a graph sense. Common "verticies" could include "alerts", "entities", "events", etc.

Extends

Scot::Model

Consumed Roles

    Meerkat::Role::Document
    Scot::Role::Hashable

Attributes

when

when it linked

vertices

Array of Hashref of structure: { type => collection_name, id => int_id, }

weight

A numerical score on the link. In case you want one.

context

brief explanation for why your are linking

memo

array that matches vertices, but contains mnemoic information about each vertex

[ 'memo string for v0', 'memo string for v1' ]
