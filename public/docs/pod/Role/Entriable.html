Name

Scot::Role::Entriable

Description

This role, when consumed by a Scot::Model, signifies that the object may have entries associated with it. It also provides the following attribues:

Synopsis

    if ( $obj->does_role('Scot::Role::Entriable') {
        say "Object has ".$obj->entry_count." entries posted";
    }

Attributes

entry_count

The integer number of entries associated with this object. Note: the controller code must update this count with entries are added, or deleted.
