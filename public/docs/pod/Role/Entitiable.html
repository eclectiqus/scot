Name

Scot::Role::Entitiable

Description

This role, when consumed by a Scot::Model, signifies that this object can have Entities associated it. This allows for code to check by doing:

Synopsis

    if ( $obj->does_role('Scot::Role::Entitiable') ) {
        print "This object can have entities in it\n";
    }
